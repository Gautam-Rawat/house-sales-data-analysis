# 🏡 House Sales Data Analysis — King County, USA

## 📌 Project Overview
This project explores and analyzes housing sales data from King County, USA to uncover key factors that influence property prices.

The goal is to perform **Exploratory Data Analysis (EDA)** and derive actionable insights that could help buyers, sellers, and real estate analysts make informed decisions.

---

## 🎯 Objectives
- Clean and preprocess raw housing data  
- Perform exploratory data analysis (EDA)  
- Identify key features affecting house prices  
- Visualize trends and relationships in the dataset  
- Generate meaningful business insights  

---

## 🛠️ Tech Stack
- **Python**
- **Pandas** — Data manipulation  
- **NumPy** — Numerical operations  
- **Matplotlib & Seaborn** — Data visualization  
- **Jupyter Notebook**

---

## 📊 Key Analysis Performed
- ✅ Data Cleaning (handling missing values, formatting)  
- ✅ Feature Exploration and Statistical Summary  
- ✅ Correlation Analysis  
- ✅ Price Distribution Analysis  
- ✅ Relationship between house features (e.g., area, bedrooms, location) and price  
- ✅ Visualization of trends using plots and charts  

---

## 🔍 Key Insights
- Larger living areas significantly impact house prices  
- Location plays a crucial role in price variation  
- Certain features (like waterfront view, condition, and grade) strongly influence value  
- Price distribution is right-skewed, indicating presence of high-value outliers  

---

## 📁 Dataset
- **Dataset:** King County House Sales Data  
- Contains information about house prices, features, and locations
